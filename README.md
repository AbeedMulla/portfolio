# Abeed Mulla — Portfolio Website

An interactive, single-page portfolio built with **React**, featuring smooth section navigation, project case-study pages, a custom animated background system, and a robot mascot UI.

> Live Link: 

---

## ✨ Features

- **Single-page layout** with smooth scrolling sections (About, Skills, Projects, Experience, Contact)
- **Hash-based project pages** (ex: `#project/interntracker`) for detailed case studies
- **Dark / Light mode** with saved preference
- **Contextual background morphing** that changes per section for a “living UI”
- **Robot mascot UI** with subtle animations and interactive elements
- **Responsive design** (works on desktop + mobile)
- **Contact modal** with basic validation + accessible form attributes

---

##  Tech Stack

- **React**
- **Vite**
- **Tailwind CSS**
- **Lucide Icons**
- **CSS-in-JS / inline styling** (clean, component-contained styling)
- **Web APIs**: `IntersectionObserver`, `localStorage`, `matchMedia` (prefers-reduced-motion)
- **Google Fonts** (Sora + JetBrains Mono)
